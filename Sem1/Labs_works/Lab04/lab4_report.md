## 2. Задание

В соответствии с индивидуальным вариантом требуется:

1. Вычислить значения функции, представленной степенным рядом:

S(x) = Σ[k=0..∞] ( x^k * (k+1) / k! )


2. Вычислить библиотечное значение:

F(x) = e^x * (1 + x)


3. Для каждого x определить:

- S(x)  
- F(x)  
- Δ(x) = |S(x) – F(x)|  
- количество членов ряда N  

4. Сформировать Таблицу 1 на интервале `[a; b]` с шагом `h`.  
5. Сформировать Таблицу 2 для фиксированного `x0` при разных ε.

---

## 3. Данные варианта

**Степенной ряд**

S(x) = Σ[k=0..∞] ( x^k * (k+1) / k! )


**Библиотечная функция**

F(x) = e^x * (1 + x)


**Начальный член**

a0 = 1


**Рекуррентная формула**

a(k+1) = a(k) * x * (k+2) / (k+1)^2


**Условие остановки**

|a(k+1) − a(k)| < ε


---

## 4. Решение

Разработана программа на C++, которая:

- вычисляет сумму степенного ряда рекуррентно,
- использует критерий остановки `|a(k+1) − a(k)| < ε`,
- сравнивает S(x) с F(x) = exp(x)*(1+x),
- формирует Таблицу 1 и Таблицу 2.

Код программы пользователь вставляет самостоятельно.
